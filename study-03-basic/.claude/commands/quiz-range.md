# 퀴즈 범위 검토: $1번 ~ $2번

$1번부터 $2번까지의 퀴즈 문제를 검토하고, 난이도와 정답 분포를 분석해 주세요.

## 검토 대상

- **시작 문제 번호**: $1
- **끝 문제 번호**: $2

## 수행 작업

### 1. 문제 수집
- 현재 디렉토리(및 하위 디렉토리)에서 퀴즈 파일을 찾습니다.
- $1번부터 $2번까지의 문제를 추출합니다.
- 해당 범위의 문제가 없거나 일부만 존재하면 명확히 보고합니다.

### 2. 난이도 분석
각 문제에 대해 다음 기준으로 난이도(상/중/하)를 판정합니다.

- **하(쉬움)**: 일반 상식 수준, 한 번에 답이 떠오르는 문제
- **중(보통)**: 약간의 사고나 배경 지식이 필요한 문제
- **상(어려움)**: 전문 지식, 추론, 복합적 판단이 필요한 문제

판정 근거를 한 줄로 제시합니다.

### 3. 정답 분포 분석
- 객관식 문제: 보기별 정답 개수와 비율(예: 1번 30%, 2번 20% ...)을 집계합니다.
- 주관식/단답형: 별도로 개수를 표시합니다.
- 특정 보기에 정답이 편중되어 있으면 경고합니다(이상적 분포는 균등에 가까움).

### 4. CLAUDE.md 검증 항목 점검
프로젝트 CLAUDE.md의 "퀴즈 문제 교차 검증 가이드라인"에 따라 각 문제를 점검합니다.

1. 정답이 하나뿐인가?
2. 최상급 표현(가장, 최초, 최대 등)에 측정 기준이 있는가?
3. 시간과 범위가 명확한가?
4. 교차 검증이 필요한 의심 정보가 있는가?

문제가 있는 항목은 문제 번호와 함께 지적합니다.

```

## 문제별 상세

| $1   | 중     | 2    | 없음       |
| ...  | ...    | ...  | ...       |

## 분포 평가
- 난이도 균형: (적절/편중)

## 검증 이슈 목록
- [문제 번호] 이슈 내용

## 권장 개선 사항
- ...
```
